# LiveEvent

A Private Live Poll and Word Cloud System with an integrated Prize-Spinning Wheel. Designed for interactive event engagement and custom branding.

## Features
- **Real-time Word Cloud**: Instant visualization of audience feedback.
- **Integrated Lucky Draw**: Randomly pick winners from participants.
- **Customizable UI**: Styled for premium event experiences.
